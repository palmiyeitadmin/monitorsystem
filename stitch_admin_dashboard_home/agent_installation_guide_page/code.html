Prerequisites

  • Windows Server 2012 R2 or newer, or Windows 10/11.
  • Administrator privileges.
  • .NET Framework 4.8 or later.

Installation Steps

  1. Download & Run Installer:

    Download the Windows agent from the table above and run the `.msi` installer as an administrator.

  2. Enter Configuration:

    When prompted, enter your unique API Endpoint and API Key.

    https://api.eramonitor.com
    your-unique-api-key-goes-here-abc123
  3. Select Services to Monitor:

    Choose the services (e.g., IIS, SQL Server) you want the agent to monitor from the checklist.

  4. Complete Installation:

    Follow the remaining prompts to complete the installation. The `ERAMonitorAgent` service will start automatically.

Verify Installation

Open PowerShell and run the following commands to check the agent's status:

# Check if the service is running
Get-Service -Name ERAMonitorAgent

# View the latest agent logs
Get-Content -Path "C:\Program Files\ERAMonitor\agent.log" -Tail 10
